# SplitFiles

A text splitter that splits a large text file into several smaller text files by line.

一个文本分割器，可以将大文本文件按行分割成若干小文本文件。

## 介绍

这是一个简单的文本分割器，可以将大文本文件**按行**分割成若干小文本文件。
拥有简单的 GUI 界面

因为我找了一圈没发现简单好用的能实现同需求的分割器，所以干脆自己写了一个，特此分享出来。

### 效率与特点

1. 对中文与特殊符号友好
2. 效率高，速度快
